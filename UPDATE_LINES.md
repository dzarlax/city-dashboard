# Quick Guide: Updating Line Data

## Когда нужно обновлять

- Добавились новые линии
- Изменились маршруты (улицы)
- Изменилась категория линий
- Нет ссылки на линию при клике

## Быстрое обновление

```bash
# 1. Скачать новый файл lines с BG Prevoz
# 2. Положить в корень проекта (вместо старого)
# 3. Запустить
npm run update-lines
```

## Что делает скрипт

✓ Генерирует `line_mapping.json` - для кликабельных ссылок
✓ Генерирует `line_details.json` - для tooltip с маршрутами
✓ Создаёт оба варианта: кириллица + латиница
✓ Парсит улицы из маршрутов

## Результат

- 643 линий в маппинге
- 578 линий с улицами
- В среднем 17 улиц на линию
- Поддержка обоих направлений (A и B)

## Проблемы?

Если линия не кликается:
1. Проверьте что `lines` файл обновлен
2. Запустите `npm run update-lines`
3. Проверьте консоль браузера на ошибки
